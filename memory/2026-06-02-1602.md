## 1. The Agent Skill Stack (Install These First)

HyperFrames ships **5 official skills** that turn Claude Code (or Cursor/Codex/Gemini CLI) into a motion director. Install with one command:

```bash
npx skills add heygen-com/hyperframes

| Skill | Role | Slash Command |
|-------|------|---------------|
| **hyperframes** | Core composition authoring — HTML structure, GSAP timelines, captions, transitions, audio-reactivity | `/hyperframes` |
| **hyperframes-cli** | Dev-loop commands — `init`, `lint`, `inspect`, `preview`, `render`, `doctor` | `/hyperframes-cli` |
| **hyperframes-registry** | Catalog block installer — knows how to wire `hyperframes add` commands into your project | `/hyperframes-registry` |
| **gsap** | Animation API reference — timelines, easing, plugins, ScrollTrigger, MotionPath | `/gsap` |
| **website-to-hyperframes** | Full URL-to-video pipeline — captures screenshots, extracts design tokens, writes `DESIGN.md`, `SCRIPT.md`, `STORYBOARD.md`, and renders | `/website-to-hyperframes` |

**Pro tip:** Always prefix prompts with `/hyperframes`. Without it, the agent invents generic web-video conventions that break the framework.

## 4. Shader Transitions: The 14 Cinematic Options

This is where HyperFrames separates from basic CSS animation. These are **WebGL pixel-shader transitions** composited frame-by-frame via HyperShader.

| Energy Level | Shaders |
|--------------|---------|
| **Calm / Editorial** | `cross-warp-morph`, `light-leak`, `domain-warp` |
| **Medium / Professional** | `cinematic-zoom`, `whip-pan`, `sdf-iris` |
| **High / Aggressive** | `glitch`, `chromatic-split`, `ridged-burn` |
| **Ethereal / Mysterious** | `gravitational-lens`, `ripple-waves`, `swirl-vortex` |
| **Utility** | `flash-through-white`, `thermal-distortion` |

**Professional rule:** ~95% of cuts in pro video are hard cuts. Reserve shaders for **2–3 hero moments** per video — the product unveil, the energy shift, the CTA landing. A 6–8 scene promo wants **2 shader transitions** max; the rest should be hard cuts with entrance animations doing the visual work.

## 5. Creative Use Cases & Community Inspiration

These are the most inventive implementations found in the wild:

### A. Website-to-Video Pipeline (The "Vibe-Coding" Workflow)
A single URL becomes a fully rendered promo. The `website-to-hyperframes` skill:
1. Captures scroll screenshots, assets, fonts, and color tokens
2. Generates `DESIGN.md` with exact HEX codes from the site's CSS
3. Writes `SCRIPT.md` with hook → story → proof → CTA structure
4. Produces `STORYBOARD.md` with per-beat mood, camera, transitions, and asset audit
5. Renders with real narration timing from TTS transcription

**Real example:** Web Reactiva used this to turn their own homepage into a 15-second brand reel with zoom-through transitions, SVG path drawing, and kinetic typography — rendered in 3 min 33 sec with 6 parallel workers.

### B. Audio-Reactive Motion Graphics
Map frequency bands to visual properties for music-driven promos:

| Audio Band | Visual Property | Effect |
|------------|----------------|--------|
| Bass | `scale` | Text/logo pulses on the beat |
| Treble | `glow` | Shimmer intensity |
| Amplitude | `opacity` | Breathing/fade |
| Mids | `shape` | Morphing geometry |

**Prompt vocabulary:** *"Make the text pulse with the beat"* or *"Add bass-driven scale to the logo"* — the agent auto-wires the GSAP adapter.

### C. Hand-Drawn Marker Highlights
Add organic emphasis effects to key phrases:

| Mode | Effect | Best For |
|------|--------|----------|
| `highlight` | Marker sweep | Key phrases |
| `circle` | Hand-drawn ellipse | Single words |
| `burst` | Radiating lines | Hype moments |
| `scribble` | Chaotic scratch | Cross-outs / corrections |
| `sketchout` | Rectangle outline | Callouts |

### D. The "Sound-Off" Social Format
Since Kokoro TTS (the local, no-API voice engine) is English-only, Spanish-speaking creators pivoted to **caption-as-narration** Reels-style videos. Kinetic captions carry the entire story. This is now a deliberate creative format, not a limitation.

### E. PR Walkthroughs with Animated Code Diffs
HyperFrames is used at HeyGen and by teams like **tldraw** and **TanStack** to produce:
- Animated code diffs with narration
- Feature announcement videos with exact UI screenshots
- Release notes turned into 30-second explainers

### F. Batch Social Clip Factories
Build one master template in HyperFrames, feed it a CSV of product features/testimonials/stats, and render batch variations. Combine with CapCut or Captions.ai for platform-specific polish.

### G. Seven-Step Production Pipeline (HeyGen's Internal Method)
Every production-grade launch video at HeyGen follows this artifact chain:

```
Capture → Design → Script → Storyboard → VO + Timing → Build → Validate
```

| Step | Output | What It Does |
|------|--------|--------------|
| 1. Capture | `capture/` | Screenshots, design tokens, assets from source URL |
| 2. Design | `DESIGN.md` | Brand cheat sheet: colors, typography, components, do's/don'ts |
| 3. Script | `SCRIPT.md` | Narration backbone: hook, story, proof, CTA |
| 4. Storyboard | `STORYBOARD.md` | Per-beat creative direction: mood, camera, techniques, transitions, SFX |
| 5. VO + Timing | `narration.wav` + `transcript.json` | TTS + word-level timestamps = source of truth for scene durations |
| 6. Build | `compositions/*.html` | One HTML file per beat, wired into `index.html` |
| 7. Validate | Snapshots + lint/validate | Visual regression tests before delivery |
