# AI Work Output Presentation

Single-page React presentation for **"10 Sites Knowledge Workers Should Build with AI"**.

## Run

```bash
cd ~/.openclaw/workspace/presentations/2026-06-11-ai-work-output
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Content Editing

All presentation copy and section data lives in:

- `src/config/site.ts`

No layout edits required for standard content updates.
