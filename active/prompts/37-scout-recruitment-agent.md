=== SCOUT RECRUITMENT AGENT — Build Prompt ===

Build an AI recruiting agent that sources, screens, and schedules candidates automatically.

PRODUCT: ScoutAI — "Your AI recruiter, working 24/7"

CORE FEATURES:
- Job intake: user describes role, Scout generates ideal candidate profile + job post
- Sourcing: searches LinkedIn, GitHub, AngelList, niche job boards automatically
- Filtering: ranks candidates by fit score based on skills, experience, location, company history
- Enrichment: scrapes GitHub repos (engineers), portfolios (designers), blog posts, Twitter
- Outreach: generates personalized messages, A/B tests subject lines, sends via LinkedIn/email
- Follow-ups: automatic 3-4 touch sequences with smart timing
- Screening: async text/voice interviews with qualifying questions
- Scheduling: finds availability, books interviews on hiring manager's calendar
- Pipeline: Kanban board tracking candidates through stages

TECH STACK:
- Sourcing: LinkedIn API (or Apify scraper), GitHub API, AngelList API
- Enrichment: Clearbit, Hunter.io for contact info
- AI: Claude/GPT-4 for message generation, interview questions, candidate summaries
- Voice: ElevenLabs for async voice interviews
- Email: SendGrid or Resend
- Calendar: Google Calendar API, Calendly
- Database: PostgreSQL
- Frontend: Next.js + Tailwind
- Backend: Node.js + Express

TARGET: Hiring managers, recruiters, startup founders, agencies

PRICING:
- Starter: $99/month (5 active roles)
- Growth: $299/month (20 active roles)
- Enterprise: $999/month (unlimited + white-label)
- Per-placement: $500 bonus when candidate is hired

SUCCESS CRITERIA:
- 50+ qualified candidates sourced per role within 7 days
- 20%+ response rate on outreach (industry avg: 5-10%)
- 3+ candidates reach interview stage per role within 14 days
- Hiring manager saves 10+ hours per role
