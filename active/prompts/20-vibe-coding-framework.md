=== VIBE CODING FRAMEWORK — Concept Guide ===

FRAMEWORK: Vibe Coding

CORE IDEA:
Use natural language to describe what you want, and AI builds it. You don't need to know how to code — you just need to know what you want.

TOOLS:
- Replit — deploy apps from chat
- Lovable — AI builds full-stack apps
- Claude / Codex — write code from descriptions
- Bolt — AI coding assistant
- V0 — Vercel's AI frontend builder

HOW IT WORKS:
1. Describe your app in plain English
2. AI generates code (React, Next.js, Python, etc.)
3. Test it, give feedback
4. AI fixes and iterates
5. Deploy in minutes

EXAMPLE PROMPT:
"Build a landing page for a Google Maps scraper. Header: 'Find Every Roofer in Phoenix'. Input field for zip code. Button: 'Get Leads'. Shows results in a table with name, phone, address. Stripe payment modal for $29. Clean, modern, mobile-friendly."

RESULT: Working landing page in 30 minutes.

WHY IT'S POWERFUL:
- Build MVP in 1 afternoon
- No technical knowledge needed
- Iterate based on customer feedback instantly
- Cost: $0-20/month for AI tools
- Compare to: $10K+ for freelance developer

BEST PRACTICES:
- Start simple — one feature, not 10
- Use proven stacks (Next.js, Tailwind, Stripe)
- Test with real users before adding complexity
- Document what works for future projects

SUCCESS CRITERIA:
- Working prototype in < 4 hours
- Customer can use it without explanation
- You can modify it yourself using AI
