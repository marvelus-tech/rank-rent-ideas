=== SHOPIFY STORE SCRAPER — Build Prompt ===

Build a tool that finds and analyzes Shopify stores.

PRODUCT: StoreSpy — "Find every Shopify store in your niche"

CORE FEATURES:
- Search by: industry, estimated revenue, traffic, apps used, theme
- Scrape Shopify stores via Apify + BuiltWith data
- Store profile: revenue estimate, traffic, top products, ads running, apps installed
- Compare: side-by-side competitor analysis
- Export to CSV
- Alert: "Tell me when new store launches in pet niche"
- Chrome extension: analyze any store you're browsing

TECH STACK:
- Frontend: Next.js + Tailwind
- Backend: Node.js + Express
- Scraper: Apify Shopify store scraper + BuiltWith API
- Database: PostgreSQL
- Chrome ext: Manifest v3
- Payments: Stripe ($49/month)
- Hosting: Vercel + Railway

TARGET: Shopify app developers, e-commerce agencies, dropshipping researchers

SUCCESS CRITERIA:
- Search "pet stores doing $50K+/month" → 100 results in 10 seconds
- Revenue estimates within 30% accuracy
- Chrome extension shows store stats on any Shopify site
