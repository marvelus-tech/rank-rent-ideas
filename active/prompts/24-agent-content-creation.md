=== AGENT CONTENT CREATION — Build Prompt ===

Build an AI agent that fully executes content creation workflows end-to-end.

PRODUCT: ContentAgent — "From idea to published post, fully automated"

CORE FEATURES:
- Research mode: scans trending topics, competitor content, Reddit discussions for ideas
- Script/Thread writer: generates YouTube scripts, Twitter threads, newsletter drafts
- Thumbnail generator: auto-creates YouTube thumbnails using AI image generation
- Scheduling: auto-posts to Twitter/X, LinkedIn, YouTube, Substack
- Performance tracking: monitors views, engagement, click-through rates
- Iteration: analyzes what worked, suggests next content pieces
- Multi-format: one research source → script + thread + newsletter + short-form

TECH STACK:
- Content research: Apify (Reddit scraper, Twitter scraper) + OpenAI analysis
- Writing: Claude/GPT-4 for long-form, GPT-4o-mini for short-form
- Images: DALL-E 3, Midjourney API, or Stable Diffusion
- Video: Remotion for automated video generation
- Scheduling: Twitter/X API, LinkedIn API, YouTube Data API, Substack API
- Analytics: Native platform APIs + dashboard
- Database: PostgreSQL (content calendar, performance data)

TARGET: YouTubers, newsletter writers, Twitter/X creators, podcasters, marketing agencies

DIFFERENTIATOR:
- Not just a writing tool — actually posts for you
- Learns from performance data (what your audience engages with)
- Handles full workflow: research → write → create assets → publish → analyze

SUCCESS CRITERIA:
- Publishes 5+ pieces/week without human intervention
- Engagement rate improves 20%+ within 60 days
- Creator saves 10+ hours/week
