=== LANGUAGE LEARNING AGENT — Build Prompt ===

Build an AI voice tutor for conversational language practice.

PRODUCT: TalkTutor — "Fluent in 3 months, actually"

CORE FEATURES:
- Voice-first conversations: AI speaks in target language, user responds by voice
- Adaptive difficulty: adjusts to user's level in real-time
- Scenario practice: roleplays (ordering food, job interview, making friends)
- Pronunciation correction: compares user's speech to native, gives feedback
- Progress tracking: vocabulary mastered, fluency score, conversation hours
- Cultural notes: explains idioms, customs, context beyond just words
- Multi-language: Spanish, French, German, Japanese, Mandarin, Russian, etc.
- Offline mode: download lessons for practice without internet
- Integrations: connects to podcasts, YouTube videos for immersive learning

TECH STACK:
- Voice AI: ElevenLabs (native-quality voices in 30+ languages)
- Speech recognition: Whisper (OpenAI) for transcription + pronunciation analysis
- Conversation AI: Claude/GPT-4 with language-teaching prompts
- Pronunciation scoring: Custom algorithm comparing phoneme alignment
- Content: Structured lesson plans (A1-C2 levels)
- Mobile app: React Native or Flutter
- Backend: Node.js + PostgreSQL + Redis
- Payments: Stripe ($19/month subscription)

TARGET: Language learners (Duolingo graduates wanting fluency), corporate training, travelers

DIFFERENTIATOR:
- Not just vocab drills — actual conversation practice
- Pronunciation feedback (most apps don't do this well)
- Available 24/7 (no scheduling tutors)
- 1/10th the cost of human tutors

SUCCESS CRITERIA:
- User has 10+ minute conversation within first week
- Pronunciation score improves 20%+ within 30 days
- User reports feeling "more confident speaking" within 2 weeks
- 50%+ monthly retention
