=== GEELONG LIFESTYLE DIRECTORY — Build Prompt ===

Build a hyperlocal lifestyle directory for Geelong, Victoria.

PRODUCT: GeelongGuide — "The best of Geelong, curated by locals"

CORE FEATURES:
- Business listings: food, fitness, entertainment, services with photos, hours, menus
- Curated collections: "Best coffee in Geelong West", "Hidden brunch spots", "Dog-friendly pubs"
- Events calendar: local events, markets, live music, pop-ups
- User reviews: community-driven ratings and photos
- Deals/offers: local businesses post limited-time offers
- Map view: interactive map with filters by category
- Newsletter: weekly "What's On" email to subscribers
- Mobile app: progressive web app for on-the-go discovery

TECH STACK:
- Frontend: Next.js + Tailwind + Mapbox
- Backend: Node.js + Express
- Database: PostgreSQL + PostGIS for location queries
- CMS: Strapi or Sanity for business profiles
- Maps: Mapbox or Google Maps API
- Reviews: Custom system or Disqus
- Newsletter: Mailchimp or ConvertKit
- Hosting: Vercel + Railway

TARGET: Geelong residents, visitors, local business owners

MONETIZATION:
- Free listings for basic profiles
- Featured listings: $49/month (top placement, photos, menu)
- Sponsored collections: $199 (curated list featuring 5-10 businesses)
- Event promotion: $29/post
- Newsletter ads: $99/issue

SUCCESS CRITERIA:
- 100+ businesses listed within 30 days
- 1000+ monthly active users within 90 days
- 10+ paying featured listings within 60 days
