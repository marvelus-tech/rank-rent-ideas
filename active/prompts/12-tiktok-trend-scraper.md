=== TIKTOK TREND SCRAPER NEWSLETTER — Build Prompt ===

Build a weekly trend report tool for TikTok marketers.

PRODUCT: TrendRadar — "Know TikTok trends before your competitors"

CORE FEATURES:
- Scrape TikTok weekly: trending sounds, hashtags, creators
- Filter by niche: beauty, fitness, food, finance, etc.
- Weekly email report: top 10 trends + how to use them
- Creator spotlights: fast-rising micro-influencers
- Sound library: trending audio clips with usage stats
- Hashtag tracker: volume + velocity (trending up/down)
- Archive: past trends with performance data

TECH STACK:
- Frontend: Next.js + Tailwind
- Backend: Python FastAPI
- Scraper: Apify TikTok scraper
- Email: Mailchimp API or SendGrid
- Database: PostgreSQL
- Payments: Stripe ($49/month for agencies, $19/month for creators)
- Hosting: Railway + Vercel

TARGET: Marketing agencies, content creators, brands with TikTok strategy

SUCCESS CRITERIA:
- Weekly email sent every Monday 8am
- 90% of trends identified before they peak
- Creator recommendations lead to 2x engagement for users
