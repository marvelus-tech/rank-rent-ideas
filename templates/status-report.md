# Structured Status Report Template

**Use this for ALL status updates. No more "things look good."**

```
## Status Report: [Task/System Name]
**Timestamp:** [ISO datetime]
**Reporter:** [Agent name]

### Summary
[One sentence: what state is this in?]

### Checks
| Item | Status | Detail |
|------|--------|--------|
| [Check 1] | ✅ PASS / ⚠️ WARN / ❌ FAIL | [Specific detail] |
| [Check 2] | ... | ... |

### What Happened
[Bullet list of actions taken]

### What Matters
[Why should the human care? Skip if nothing actionable]

### What's Next
[Concrete next steps, if any]

### Confidence: [0-100]
### Evidence
[Links, hashes, file paths, output snippets]
```
