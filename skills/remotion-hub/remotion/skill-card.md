## Known Risks and Mitigations: <br>
Risk: Package-manager examples can change a project dependency set if copied directly. <br>
Mitigation: Review package additions before running commands and use trusted package names and registries. <br>
Risk: Remote media and fetch examples can make network requests during preview or render. <br>
Mitigation: Prefer local or trusted remote assets, and review external URLs before rendering. <br>
