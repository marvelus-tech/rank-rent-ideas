## Description: <br>
Helps agents create frontend UI code for webpages, landing pages, dashboards, React/Vue components, and other frontend pages with stronger typography, color, motion, background, and layout direction. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[moxunjinmu](https://clawhub.ai/user/moxunjinmu) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers and design-focused agents use this skill to generate polished HTML/CSS/JavaScript or React/Vue interface code with distinctive visual direction. It is not intended for static image design or WeChat graphic assets. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Generated pages may include external frontend dependencies such as Google Fonts or Framer Motion, which can create privacy, availability, or supply-chain concerns. <br>
Mitigation: Review generated code before use, approve or pin dependencies, and vendor assets locally when project policy requires it. <br>
Risk: Strong visual constraints around typography, motion, color, and asymmetric layout may conflict with accessibility, brand, or product usability requirements. <br>
Mitigation: Check generated interfaces for accessibility, responsive behavior, brand fit, and motion sensitivity before deployment. <br>
